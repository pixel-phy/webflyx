# Webflyx

Es un proyecto educativo desarrollado como parte del backend roadmap en [Boot.dev](https://boot.dev). El objetivo principal es comprender la arquitectura interna de Git, el manejo de repositorios locales y remotos, y los fundamentos del desarrollo backend.

## 🚀 Características
* Gestión de datos simulada en el entorno local.
* Conexión fluida con repositorios remotos en GitHub.
* Estructura limpia basada en los estándares modernos de Git (`main` branch).

## 🛠️ Tecnologías utilizadas
* **Git & GitHub**: Control de versiones y alojamiento en la nube.
* **Linux/Unix CLI**: Gestión de archivos y automatización desde la terminal.
